# Jan Pavlovič

Aplikace zpracovává CSV soubory a ZIP archivy ze složky `input` a ukládá výsledky do složky `output`.
Pomocí třídy `Fraction` sjednocuje formát dat, sčítá zlomky i procenta a automaticky je krátí.
Podporuje různé oddělovače sloupců (`,`, `;`, `:`, `|`, `=`, `\`) a zachovává původní názvy souborů.
Ke spuštění stačí umístit vstupní data do složky a spustit hlavní třídu `Main`.
